&brandFullName; 사용하기

&brandFullName; 사용을 환영합니다! &brandShortName;는 웹 페이지를 방문하고 검색하는데 사용하는 인터넷 브라우저입니다.

세부 항목:
  • 웹 페이지 탐색하기
  • 검색하기
  • 복사하기, 저장하기 및 페이지 인쇄하기
  • 속도와 효율 개선하기

웹 페이지 탐색하기

사용자 홈 페이지 보기

&brandShortName;를 시작하면, &brandShortName;의 시작 페이지를 보게 될 것입니다. 시작 페이지는 빈 페이지, 홈 페이지, 이전 종료시 창과탭 중 하나입니다. 기본적으로, 시작 페이지는 홈 페이지로 지정되어 있을 것입니다. 시작 페이지나 홈 페이지를 변경하려면 &pref.menuPath; 메뉴를 선택하고 일반 설정을 클릭합니다.

유용한 정보:

  • 화면에 더 많은 내용을 표시하려면, 전체 화면 모드를 사용할 수 있습니다. 전체 화면 모드에서는 &brandShortName;의 도구 모음이 작게 축소됩니다. 전체 화면 모드를 활성화 하려면, 간단하게 보기 > 전체 화면을 클릭하거나 F11 누릅니다.
  • 사용자의 홈 페이지로 바로 이동하려면, &altKey;+Home를 누릅니다.

다른 페이지로 이동하기

주소 표시줄에 인터넷 주소나 URL을 입력하면 해당 웹 페이지로 이동할 수 있습니다. URL은 보통 "http://"로 시작하여 해당 주소를 식별할 수 있는 이름이 따라옵니다. 예를 하나 들자면, "http://www.mozilla.org/"와 같습니다.

  1. 주소 표시줄을 클릭하여 이미 존재하는 URL을 하나 선택합니다.
  2. 방문하고자 하는 페이지의 URL을 입력합니다. 주소 표시줄에 남아 있던 텍스트는 새로 입력한 URL로 대체됩니다.
  3. &enterKey;를 누릅니다.

유용한 정보: 주소 표시줄에서 URL을 빠르게 선택하려면, &accelKey;+L를 누릅니다.

URL를 모르세요? 주소 표시줄에 방문하려는 페이지에 대한 구체적인 단어-예를 들면, 이름-를 입력하고, &enterKey;를 누릅니다. 해당 검색어에 대한 구글의 첫번째 검색 결과로 이동할 것입니다.

링크 클릭하기

대부분의 웹 페이지는 링크를 포함하고 있으며, 이 링크를 클릭하여 다른 페이지로 이동할 수 있습니다.

  1. 마우스 포인터를 움직여서 포인터가 손가락 모양이 되게 합니다. 포인터가 링크 위에 있을 때마다 손가락 모양이 됩니다. 대부분의 링크는 밑줄이 있는 텍스트이지만, 단추이나 그림도 링크가 될 수 있습니다.
  2. 링크는 단 한 번만 클릭합니다. 네트워크가 링크 페이지를 찾아가는 동안, 상태 메시지가 창 하단에 나타날 것입니다.

열어본 페이지로 되돌아 가기

여러 가지 방법으로 페이지를 재 방문할 수 있습니다:

  • 한 페이지 뒤로나 앞으로 이동하려면, 뒤로 또는 앞으로 단추를 클릭합니다.
  • 한 페이지 이상으로 뒤로나 앞으로 이동하려면, 뒤로나 앞으로 단추에 있는 작은 삼각형을 클릭합니다. 최근 방문한 페이지 목록이 나타날 것입니다; 목록에서 원하는 페이지를 선택합니다.
  • 주소 표시줄에 직접 입력했던 URL 목록을 보려면, 주소 표시줄 오른쪽 끝에 있는 화살표를 클릭합니다. 원하는 페이지를 목록에서 선택합니다.


  • 현재 세션에서 방문했던 페이지를 선택하려면, 바로 가기 메뉴를 열어서 메뉴 아랫 부분에 있는 목록을 이용합니다.
  • 지난 여러 세션에서 방문했던 페이지를 선택하려면, 바로 가기 메뉴를 열어 탐색 창에서 보기을 선택합니다. 열어본 페이지 목록이 탐색 창이 나타날 것입니다. 열어본 페이지 목록 탐색 창에 폴더 목록이 나타납니다. 폴더를 클릭하면 하부 폴더나 웹 페이지 제목이 나타납니다. 페이지 제목을 클릭하면 해당 페이지를 볼 수 있습니다.

중지하기 및 새로 고치기

페이지가 너무 느리가 열리거나 더 이상 보고 싶지 않으면, 중지 단추를 클릭합니다.

현재 페이지를 다시 열거나 최신 상태를 불러 오려면, 새로 고침 단추를 클릭 하거나 &accelKey;+R를 누릅니다.

탭 브라우징

동시에 하나 이상의 웹 페이지를 방문하려고 할 때, 탭 브라우징을 사용하여 웹을 더 빠르고 쉽게 탐색할 수 있습니다.

탭 브라우징을 통해 하나의 &brandShortName; 창에 여러 탭을 열고 각 탭마다 웹 페이지를 불러 올 수 있습니다. 여러 웹 페이지를 방문하기 위해 여러 창을 열 필요가 없습니다. 이로 인해 바탕화면 공간이 절약됩니다. 한 장소(창)에서 여러 웹 페이지를 열고, 닫고, 다시 불러 올 수 있으므로 다른 창으로 옮겨 다닐 필요가 없습니다.

더 자세한 정보는 탭 브라우징을 참조합니다.

탐색 창 사용하기

탐색 창은 화면 왼쪽 영역에 위치하며 북마크나 열어본 페이지 목록을 표시합니다. 확장 기능을 통해 탐색 창을 이용하는 새로운 방법을 추가할 수도 있습니다.

창색 창 목록을 보려면, 보기 > 탐색 창을 선택합니다. 거기서 원하는 탐색 창을 선택할 수 있습니다.

검색하기

웹 검색하기

특정 주제에 대한 웹 페이지를 찾으려면 &brandShortName; 오른쪽 상단에 있는 검색 도구에 단어 몇 개를 입력하기만 하면 됩니다.

예를 들어, "baby doll"에 대한 정보를 원한다면:

  1. 검색 도구를 클릭합니다.
  2. baby doll을 입력합니다. 검색 도구에 기존에 있던 텍스트는 새로 입력하는 텍스트로 대체 됩니다.
  3. &enterKey;를 눌러 검색을 시작합니다.

"baby dool"에 대한 검색 결과가 &brandShortName; 창에 나타납니다.

검색 엔진 선택하기

검색 창 끝에 있는 화살표를 클릭하여 다른 검색 엔진을 선택할 수 있습니다. Google과 같은 검색 엔진은 전체 웹 페이지를 검색하는 반면, Amazon.com과 같은 검색 엔진은 특정 사이트만을 검색합니다.

검색 엔진 관리하기

검색 엔진 관리…를 선택하여 검색 엔진을 추가, 재정렬, 또는 삭제할 수 있습니다. 검색 엔진을 선택한 후 적당한 단추를 클릭하여 목록 내에서 위치를 이동하거나 삭제할 수 있습니다. &brandShortName; 부가 기능에서 추가적인 검색 엔진을 설치할 수 있습니다.

선택된 단어를 웹 페이지 내에서 검색하기

&brandShortName;는 선택한 단어를 해당 웹 페이지 내에서 검색할 수 있게 해 줍니다.

  1. 웹 페이지에서 단어를 선택(반전) 합니다.
  2. 오른쪽 클릭&ctrlKey;를 누르고 마우스 클릭하고 팝업 메뉴에서 [검색 엔지] 검색 : "[선택된 단어]"를 클릭합니다.

&brandShortName;가 새 탭을 열고 현재 선택된 검색 엔진을 이용하여 선택된 단어를 검색합니다.

한 페이지 내에서 검색하기

&brandShortName;에서 현재 보고 있는 페이지 내에서 텍스트를 검색하려면:

  1. 편집 > 이 페이지에서 찾기를 선택합니다. 현재 페이지에 프레임이 있으면, 검색을 시작하기 전에 해당 프레임을 클릭해야 할 것입니다. 브라우저 하단에 새로운 도구 창이 하나 표시 되어야 합니다.
  2. 찾으려는 텍스트를 입력합니다.
  3. 다음 중 하나를 선택합니다:
    • 아래로: 해당 페이지에서 현재 커서 위치 아래 쪽 텍스트를 검색합니다.
    • 위로: 현재 커서 위치 위 쪽에 있는 텍스트를 검색합니다.
    • 모두 강조: 현재 페이지에서 검색어와 일치하는 모든 텍스트에 강조 표시됩니다.
    • 대/소문자 구분: 대/소문자 구분하여 텍스트를 검색하도록 제한합니다.
  4. 텍스트 입력 상자에 글자를 입력하면 자동적으로 검색을 시작합니다.

동일한 단어나 어구를 다시 찾으려면:

  • 편집 > 다시 찾기를 선택합니다.

복사하기, 저장하기 및 페이지 인쇄하기

페이지의 일부 복사하기

현재 페이지에서 텍스트를 복사하려면:

  1. 텍스트를 선택합니다.
  2. 메뉴에서 편집 > 붙여넣기를 선택합니다.

현재 텍스트를 다른 프로그램에 붙여넣을 수 있습니다.

현재 페이지에서 링크(URL)나 그림 링크를 복사하려면:

  1. 마우스 포인터를 링크나 그림 위로 가져갑니다.
  2. 링크나 그림 위에서 오른쪽 클릭 &ctrlKey;를 누르고 마우스 클릭 하면 팝업 메뉴가 나타납니다.
  3. 링크 주소 복사나 그림 주소 복사를 선택합니다. 그림이 링크이면, 둘 중 아무거나 선택합니다.

복사된 링크를 다른 프로그램이나 &brandShortName;의 주소 표시줄에 붙여 넣을 수 있습니다.

한 페이지의 전부나 일부 저장하기

&brandShortName;에서 한 페이지 전체를 저장하려면:

  1. 파일 > 다른 이름으로 저장을 선택하면 다른 이름으로 저장 대화 상자가 나타날 것입니다.
  2. 페이지가 저장될 위치를 선택합니다.
  3. 페이지가 저장될 파일 형식을 선택합니다.:
    • 웹 페이지, 전체: 그림을 포함하여 웹 페이지 전체를 저장합니다. 이렇게 저장하면 웹 페이지를 원래 그림이 보이던 모양과 동일한 모양으로 볼 수는 있지만, 원본 페이지에 있던 HTML 링크 구조는 보존되지 않을 수도 있습니다. &brandShortName;는 페이지가 저장된 위치에 새로운 디렉토리를 생성하여 그림을 비롯하여 전체 페이지를 보여주는데 필요한 기타 파일을 저장합니다.
    • 웹 페이지, HTML만: 그림은 포함하지 않고 원본 페이지를 저장합니다. 이렇게 저장하면 원본 HTML 링크 구조가 단일 파일에 보존됩니다.
    • 텍스트 파일: 원본 페이지를 텍스트 파일로 저장합니다. 이렇게 저장하면 원본 HTML 링크 구조가 보존되지는 않지만, 해당 웹 페이지의 텍스트 버전을 아무 텍스트 편집기를 통해서나 볼 수 있습니다.
  4. 페이지가 저장될 파일 이름을 입력하고 저장을 클릭합니다.

웹 페이지 내에 있는 프레임을 저장하려면:

  1. 마우스 포인터를 프레임 내로 가져갑니다.
  2. 프레임 위에서 &ctrlKey;를 누르고 마우스 클릭 오른쪽 클릭하면 팝업 메뉴가 나타납니다.
  3. 하부 메뉴에서 현재 프레임 > 프레임을 다른 이름으로 저장을 선택합니다. 다른 이름으로 저장 대화 상자가 나타날 것입니다.
  4. 페이지가 저장될 위치를 선택합니다.
  5. 페이지가 저장될 파일 형식을 선택합니다.
  6. 페이지가 저장될 파일 이름을 입력하고 저장을 클릭합니다.

파일을 하드 디스크에 저장하면, 인터넷에 접속하고 있지 않은 때에도 해당 페이지를 볼 수 있습니다.

웹 페이지에서 그림을 저장하려면:

  1. 마우스 포인터를 그림 위로 가져갑니다.
  2. 그림 위에서 오른쪽 클릭 &ctrlKey;를 누르고 마우스 클릭하여 팝업 메뉴를 엽니다.
  3. 그림을 다른 이름으로 저장을 선택합니다. 다른 이름으로 저장 대화 상자가 나타날 것입니다.
  4. 저장될 그림의 위치를 선택합니다.
  5. 저장될 그림의 이름을 입력하고 저장을 클릭합니다.

페이지를 보지 않고 않고 저장하려면 (표시 목적이 아닌 구조화 되지 않은 페이지를 저장하는데 유용합니다):

  1. 마우스 포인터를 해당 페이지 링크 위로 가져갑니다.
  2. 링크 위에서 오른쪽 클릭 &ctrlKey;를 누르고 마우스 클릭하여 팝업 메뉴를 엽니다.
  3. 링크를 다른 이름으로 저장을 선택합니다. 다른 이름으로 저장 대화 상자가 나타날 것입니다.
  4. 저장될 페이지의 위치를 선택합니다.
  5. 저장될 페이지의 이름을 입력하고 저장을 클릭합니다.

중요: 어떤 링크는 클릭하면 자동적으로 파일을 다운로드하여 사용자의 하드 디스크에 저장합니다. 이런 링크에 대한 URL은 흔히 "ftp"로 시작하거나 "au" 나 "mpeg"과 같은 파일 확장자로 끝납니다. 이런 링크는 소프트웨어, 사운드, 또는 영화 파일을 전송하고 해당 파일을 지원하는 보조 응용 프로그램을 실행 시킬수도 있습니다.

유용한 정보: 그림을 바탕 화면 배경으로 지정하려면, 그림 위에서 오른쪽 클릭 &ctrlKey;를 누르고 마우스 클릭하고 팝어 메뉴에서 배경 화면으로…를 선택합니다.

페이지 인쇄하기

현재 페이지를 인쇄하려면:

  • 파일 > 인쇄를 클릭합니다.

선택된 텍스트를 인쇄하려면:

  • 현재 페이지에서 텍스트를 선택합니다.
  • 파일 > 인쇄를 선택합니다. 인쇄 대화 상자가 나타날 것입니다.
  • 인쇄 범위 아래에 있는 선택 영역을 클릭합니다.

웹 페이지 제작자와 (화면상 창의 크기가 아니라) 인쇄되는 페이지의 크기가 인쇄 용지 상의 내용물의 위치를 결정하게 됩니다. 텍스트는 자동 줄바꿈 되고 그림은 용지 크기에 맞추어 위치가 조정됩니다.

인쇄 미리 보기 사용하기

인쇄하기 전에 페이지가 어떻게 보일 것인지를 미리 보려면, 인쇄 미리 보기를 이용할 수 있습니다:

  • 파일 > 인쇄 미리 보기를 선택합니다.

인쇄 미리 보기에서는, 인쇄하려는 페이지에 대해 다음을 수행할 수 있습니다:

  • 각 페이지의 미리 보기: 다음 페이지 , 이전 페이지 , 첫 페이지 , 또는 마지막 페이지 단추를 클릭하여 페이지 사이를 이동합니다.
  • 페이지의 크기(축소/확대) 변경하기: 크기 드롭다운 메뉴에서, "50%" 선택하면 원본 페이지 크기의 절반으로 축소됩니다. 자동적으로 크기가 변경되어 용지 너비에 맞춰지게 하려면, "용지에 맞춤"을 선택합니다. "사용자 정의…"를 선택하면 백분율 비율을 직접 입력할 수 있습니다.
  • 페이지의 방향 바꾸기: 세로 를 선택하면 페이지가 폭이 좁은 쪽이 위로 향하게 됩니다. 가로 를 선택하면 폭이 넓은 쪽이 위로 향하게 됩니다.
  • 페이지 설정하기: 인쇄하려는 페이지에 대해 더 많은 설정을 지정하려면 페이지 설정 클릭합니다.
  • 인쇄하기: 페이지를 인쇄하려면 인쇄를 클릭합니다.

주의: 일부 인쇄 미리 보기 기능은 Mac OS와 Linux에서 다르거나 지원되지 않을 수 있습니다.

페이지 설정 사용하기

주의: 일부 페이지 설정 기능은 Mac OS와 Linux에서 다르거나 지원되지 않을 수 있습니다.

페이지가 &brandShortName;에서 어떻게 인쇄 될 것인지를 설정하려면, 페이지 설정을 이용합니다:

  • Select 파일 > 페이지 설정.

페이지 설정에서, 인쇄하려는 페이지에 대해 다음 설정을 변경할 수 있습니다:

  • 용지 및 배경: 방향, 크기, 및 기타 &pref.plural; 지정:
    • 인쇄 방향:
      • 세로: 이 모양 을 선택하면, 페이지가 일반적인 방향, 즉 폭이 좁은 쪽이 위로 향하게 됩니다.
      • 가로: 이 모양 을 선택하면, 페이지가 옆으로 누은 방향, 즉 폭이 넓은 쪽이 위로 향하게 됩니다.
    • 크기(축소/확대): 원본 크기에 대한 백분율 비율을 입력합니다. 예를 들어, "50"를 입력하면 매 페이지가 원본 페이지의 절반 크기가 됩니다.
      • 용지에 맞춤: 이를 선택하면 자동적으로 페이지의 크기가 용지 너비에 맞게 크기가 변경됩니다.
    • 배경:
      • 배경 인쇄 (색상과 그림): 이를 선택하면 배경 그림과 색상이 인쇄됩니다. 이것이 선택되어 있지 않으면 전경 그림과 색상만 인쇄됩니다.
  • 여백 및 머리글/바닥글: 이 탭을 클릭하면 여백, 머리글 및 바닥글을 설정할 수 있습니다:
    • 여백:
      • 위, 아래, 왼쪽, 오른쪽: 주어진 단위로 위, 아래, 왼쪽, 오른쪽에 대한 여백을 입력합니다
    • 머리글 & 바닥글: 각 드롭다운 리스트는 머리글이나 바닥글 중 하나에 나타납니다. 윗줄 드롭다운 리스트는 왼쪽, 가운데 및 오른쪽 머리글 영역에 대한 것입니다. 아랫줄 드롭다운 리스트는 외쪽, 가운데 및 오른쪽 바닥글 영역에 대한 것입니다. 각 드롭다운 리스트마다 다음 옵션 중 하나를 선택할 수 있습니다:
      • --공백--: 이 영역에 아무것도 표시하지 않습니다.
      • 제목: 웹 페이지 제목을 표시합니다.
      • URL: 웹 페이지 URL (URL은 보통 "http://"로 시작)을 표시합니다.
      • 날짜/시간: 웹 페이지가 인쇄되는 날짜와 시간을 표시합니다.
      • 페이지 #: 각 페이지의 페이지 번호를 표시합니다.
      • # 중 # 페이지: 페이지 번호를 전체 페이지 수와 함께 표시합니다. 예를 들어, 5쪽짜리 웹 페이지를 인쇄한다면, "5 중 3" 이 세번째 페이지에 나타날 것입니다.
      • 사용자 정의: 사용자가 직접 텍스트를 입력합니다. 다음 코드를 포함시켜 특정 정보를 인쇄할 수도 있습니다:
        • &PT: 전체 페이지 수 (예: "5 중 3")
        • &P: 페이지 번호
        • &D: 날짜
        • &U: URL
        • &T: 페이지 제목

유용한 정보: 페이지 설정에 적용된 사항을 미리 보려면, 인쇄 미리 보기를 이용합니다.

속도와 효율 개선하기

다양한 형식의 파일 관리하기

&brandShortName;는 다양한 형식의 파일을 처리할 수 있습니다. 하지만, 영화나 음악과 같은 일부 파일에 대해서, &brandShortName;는 플러그인이나 외부 응용 프로그램이 있어야만 처리할 수 있습니다. &brandShortName;가 요구되는 필러그인을 가지고 있지 않거나 어떤 응용 프로그램을 이용하여 해당 파일을 열어야할지 모르는 경우라도, 해당 파일을 사용자의 하드 디스크에 저장할 수는 있습니다. 파일을 저장한 경우에는, 다운로드 관리자를 이용하여 파일을 지속적으로 관리할 수 있습니다.

플러그인은 &brandShortName;의 기능을 확장시키며 &brandShortName;에서 실행됩니다. Sun Java, Macromedia Flash, 그리고 RealNetworks RealPlayer와 같은 플러그인은 &brandShortName;에서 멀티미디어 파일을 보여주고 영화, 에니메이션 , 그리고 게임과 같은 작은 응용 프로그램을 실행시킵니다. 흔히 쓰이는 플러그인은 모질라 업데이트 사이트의 플러그인 페이지에서 구할 수 있습니다.

파일이 &brandShortName;로 열리지 않으면, &brandShortName; 외부의 응용 프로그램을 이용할 수 있습니다. 예를 들어, 사용자가 직접 미디어 재생기를 선택하여 MP3 파일을 재생시킬 수 있습니다.

&brandShortName;로 열 수 없는 파일을 어떻게 처리할 것인지는 다운로드 실행 방식 대화 상자에서 지정할 수 있습니다. 이 대화 상자를 열려면, &pref.menuPath; 메뉴를 선택하고 내용 설정을 연 다음, 파일 형식… 단추를 클릭합니다. 파일 형식에 대한 실행 방식을 변경하거나 제거할 수 있습니다. 이 대화 상자에 대한 더 자세한 정보는 파일 형식의 해당 섹션에서 볼 수 있습니다.

파일를 클릭했는데 &brandShortName;가 어떻게 처리해야 할지를 모르면, 열기 대화 상자가 나타나 다음 중 하나를 선택토록 합니다:

  • 열기: &brandShortName;가 사용자 컴퓨터에 설정된 기본 프로그램을 이용하여 파일을 엽니다. 드롭다운 리스트에서 기타…를 선택하여 다른 프로그램을 지정할 수 있습니다.
  • 디스크에 저장: &brandShortName;가 파일을 저장할 위치를 사용자가 선택할 수 있도록 합니다. &pref.pluralCaps; 다운로드 설정에서 다음 폴더에 저장을 지정해 두면, 파일이 자동적으로 지정된 기본 폴더에 저장됩니다.
  • 다시 묻지 않음: 이 옵션을 선택하면, &brandShortName;가 자동적으로, 나중에 다시 동일한 형식의 파일을 다운로드하면, 지정된 실행 방식을 수행할 것입니다.

유용한 정보: 다시 묻지 않음 &pref.plural;이 저장된 정보는 다운로드 실행 방식 대화상자에서 초기화 할 수 있습니다.

&brandShortName;를 기본 브라우저로 지정하기

&brandShortName;는 인터넷과 사용자의 컴퓨터 상에 있는 웹 페이지를 보여줍니다. 웹 페이지를 손쉽게 열려면, &brandShortName;를 기본 브라우져로 지정할 수 있습니다.

&brandShortName;를 기본 브라우저로 지정하려면, &pref.menuPath; 메뉴를 선택하고 일반 설정에서 지금 확인 단추를 클릭합니다.

캐시 설정 변경하기

&brandShortName;는 자주 방문하는 페이지의 복사본을 저장합니다. 이를 통해 &brandShortName;는 사용자가 해당 웹 페이지를 보려고 할 때마다 네트워크로부터 이 웹 페이지를 수집할 필요가 없게 됩니다.

캐시 크기 지정하거나 비우기:

  1. &pref.menuPath; 메뉴를 선택합니다.
  2. 고급 설정을 열고 네트워크 탭을 클릭합니다.
  3. 사용 용량 필드에 캐시 크기를 숫자로 입력합니다. 50MB에서 70MB정도가 적당합니다. 캐시를 즉시 초기화하려면, 개인 정보 설정에서 지금 삭제 단추를 클릭합니다.

중요: 캐시가 크면 더 많은 자료를 빨리 수집할 수 있지만, 더 많은 하드 디스크 공간이 사용됩니다.

&brandShortName;는 종료되면서 캐시 저장을 수행합니다. 저장 시간이 너무 오래 걸리면, 디스크 캐시의 크기를 줄여 봅니다.

언제든지 페이지를 새로 고치려면 &brandShortName;의 표준 도구 모음에서 새로 고침 단추를 클릭합니다. &brandShortName;가 해당 페이지의 최신 버전을 불러와 보여 줍니다.
