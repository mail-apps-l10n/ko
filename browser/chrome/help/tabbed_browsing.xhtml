탭 브라우징

탭 브라우징을 통해 하나의 &brandShortName; 창에 여러 탭을 열고 각 탭마다 웹 페이지를 불러 올 수 있습니다. 링크를 새 탭으로 열어 페이지를 불러오는 동안 사용자는 여전히 현재 페이지를 보고 있을 수 있습니다. 탭 브라우징을 반드시 사용할 필요는 없지만, 사용해 보면 웹을 탐색하는 더 좋고, 더 빠른 방법임을 알게 될 것입니다.

세부 항목:
  • 탭 만들기 및 불러오기
  • 창 내에서 탭 이동하기
  • 탭 닫기
  • 탭 브라우징 사용자 설정하기
  • 유용한 정보

탭 만들기 및 불러오기

새로운 탭을 열려면, &accelKey;+T를 누르거나, 파일 > 새 탭을 선택하거나, 또는 탭 도구의 빈 영역을 더블 클릭합니다.

링크를 탭으로 불러오려면, 다음 중 하나를 시행합니다:

  • 링크를 마우스 가운데 클릭합니다. (마우스 휠이 있는 경우, 휠 클릭은 가운데 클릭과 동일합니다.)
  • 링크를 탭 도구의 빈 영역에 끌어다 놓습니다. (웹 페이지가 단 하나만 열려 있는 경우, 탭 도구가 숨겨져 있을 수 있습니다. 탭 브라우징 사용자 설정에서 이의 변경 방법에 대한 정보를 볼 수 있습니다.)
  • 링크를 기존 탭에 끌어다 놓으면 해당 탭에서 링크를 열 수 있습니다.
  • 링크를 오른쪽 클릭&ctrlKey;를 누르고, 마우스 클릭하고 팝업 메뉴에서 새 탭에서 링크 열기를 선택합니다.

주소 표시줄에 있는 URL를 새탭으로 열려면, URL을 입력하고 &altKey;+&enterKey; 를 누릅니다.

창 내에서 탭 이동하기

탭은 열린 순서대로 표시되는데, 때론 사용자가 원치 않는 순서일 수 있습니다. 탭을 &brandShortName; 창 내에서 다른 위치로 이동시키려면, 간단히 마우스로 탭을 끌어서 이동하면 됩니다. 탭을 끄는 동안 &brandShortName;는 탭이 이동될 위치를 보여주는 작은 지시자를 표시할 것입니다. 다른 방법으로는 키보드 단축키를 이용하여 탭을 창 내에서 원하는 곳으로 이동할 수 있습니다.

탭 닫기 및 복원하기

현재 탭을 닫으려면, &accelKey;+W를 누르거나, 탭 닫기 단추를 클릭하거나, 또는 파일 > 탭 닫기를 선택합니다. 현재 탭 이외의 모든 다른 탭을 닫으려면, 현재 탭 막대 위에서 오른쪽 클릭&ctrlKey;를 누르고, 마우스 클릭하고 탭 모두 닫기를 선택합니다.

&brandShortName;는 최근 닫은 탭 목록을 저장하고 있습니다. 닫았던 탭을 복원하려면 바로가기 > 최근에 닫은 탭을 선택합니다. 닫았던 모든 탭을 복원하려면 바로 가기 > 최근에 닫은 탭 > 모두 탭으로 열기를 선택합니다. &accelKey;+&shiftKey;+T를 누르면 하나씩 지워진 역순으로 탭으로 열 수 있습니다.

탭 브라우징 사용자 설정하기

탭 브라우징 &pref.plural;을 변경하려면 &pref.menuPath;을 선택하고 탭 설정을 엽니다.

유용한 정보

북마크와 탭 브라우징
모든 탭을 북마크 하려면:
  1. &accelKey;+&shiftKey;+D를 누르거나 북마크 > 모든 탭을 북마크에 추가를 선택하여선택하여 현재 창에 있는 모든 탭을 새로운 북마크 폴더에 추가합니다.
  2. 여러 북마크를 탭으로 열려면, 북마크 폴더 메뉴를 열고 탭으로 열기를 선택합니다. 폴더 위에서 가운데 클릭하여 그 안에 있는 북마크을 탭으로 열 수도 있습니다. 북마크는 새 탭이 아닌 기존 탭에서 열릴 것입니다.
다중 탭 홈 페이지
오직 하나의 웹 페이지만을 사용자 홈 페이지로 사용할 수 있는게 아니고, 다중 탭을 사용자 홈 페이지로 지정할 수 있습니다. 즐겨 찾는 웹 사이트들을 선별하여 한 창에서 각각 탭으로 엽니다. &pref.menuPath; 메뉴를 선택하고 일반 설정을 엽니다. 시작 페이지 섹션에서, 현재 페이지 사용 단추를 누릅니다. 이제 홈 단추를 한 번 누르기만 하면 즐겨 찾는 여러 웹 사이트를 한꺼번에 불려올 것입니다. 시작 페이지가 홈 페이지로 지정되어 있어야만, &brandShortName; 시작 시, 여기서 지정한 홈 페이지(단일 페이지이든, 다중 탭 페이지이든)가 바로 열릴 것입니다.
새 탭 도구 모음 단추 추가하기
도구 모음 사용자 설정을 통해 새 탭 단추를 도구 모음에 추가하면 손쉽게 새 탭을 열 수 있습니다.
키보드 단축키
&brandShortName;은 탭 브라우징과 관련하여 광범위한 단축키를 제공합니다. 탭 브라우징과 관련된 광범위한 단축키가 제공되며, 전체 리스트는 키보드 단축키에서 볼 수 있습니다.
