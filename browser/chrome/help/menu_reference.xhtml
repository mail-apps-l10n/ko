&brandFullName; 정보

대화 상자에 현재 버전과 간략한 공헌자 정보을 포함한 &brandShortName;에 대한 정보가 나타납니다.

설정…

&pref.pluralCaps; 창에서 &brandShortName;에 있는 &pref.plural;을(를) 변경할 수 있습니다.

&brandShortName; 종료

모든 &brandShortName; 창을 닫고, 진행 중인 모든 다운로드를 취소하고, &brandShortName;를 완전히 빠져나갑니다. 이 메뉴를 선택했는데 하나 이상의 탭이나 창이 열려 있으면, &brandShortName;는 사용자가 정말로 종료하려는지 물어 보아 실수로 종료되지 않도록 할 것입니다. &pref.menuPath; 메뉴의 탭 설정에서 여러 탭을 닫을 때 확인하기 확인란의 선택을 해제하여 이 경고 메시지가 나타나지 않게 할 수 있습니다. &brandShortName;는 또한 파일을 다운로드 하고 있을 때에도 경고 메시지를 보냅니다.

파일

새 창

새 &brandShortName; 창 하나를 엽니다.

새 탭

현재 &brandShortName; 창에서 새 탭 하나를 엽니다.

주소 열기…

&brandShortName; URL 주소 표시줄로 커서가 이동하며, 방문할 새로운 주소를 입력할 수 있습니다.

파일 열기…

파일 열기 대화 상자에서 로컬 디스크나 네트워크에서 파일을 선택할 수 있습니다. &brandShortName;로 HTML/XML뿐 아니라 그림 파일, 텍스트 파일 등 많은 종류의 파일을 볼 수 있습니다.

(창) 닫기

현재 창을 닫습니다.

탭 닫기

현재 탭을 닫고 맨 오른쪽 탭이 선택됩니다. 이 메뉴는 하나 이상의 브라우져 탭이 열려 있어냐만 보입니다.

다른 이름으로 저장…

현재 보고 있는 페이지를 저장합니다. 페이지 전체(그래픽을 모두 포함해서)을 저장할 것인지, 단지 현재 HTML 페이지만을 저장할 것인지, 아니면 그 페이지에 있는 텍스트만 저장할 것인지를 선택할 수 있습니다.

메일로 링크 보내기…

사용자 컴퓨터의 기본 전자 메일 프로그램의 메일 작성 창을 열어 현재 페이지에 대한 링크를 보낼 수 있습니다.

페이지 설정…

페이지 설정 대화 상자에서 여백, 머리글과 바닥글, 및 용지 방향과 같은 인쇄 설정을 지정할 수 있습니다.

인쇄 미리 보기

현재 문서가 인쇄 시 어떻게 보일 것인가에 대한 미리 보기가 나타납니다. 페이지 설정 대화 상자의 설정 값과 해당 페이지 작성자의 출력 의도가 반영됩니다.

인쇄…

인쇄 대화상자에서 인쇄할 사본 수 등을 지정할 수 있습니다. 확인을 클릭하여 해당 페이지를 인쇄합니다.

가져오기…

가져오기 마법사를 통해 &pref.plural;, 북마크, 열어본 페이지 목록, 비밀 번호 및 기타 자료를 Microsoft Internet Explorer, Netscape, Mozilla 또는 Opera와 같은 다른 브라우저에서 가져올 수 있습니다.

오프라인으로 작업

온라인 모드와 오프라인 모드 사이를 전환할 수 있습니다. 오프라인 모드에서는 인터넷에 접속하지 않고도 이전에 방문했던 페이지를 볼 수 있습니다.

종료 종료

모든 &brandShortName; 창을 닫고, 진행 중인 모든 다운로드를 취소하고, &brandShortName;를 완전히 빠져나갑니다. 이 메뉴를 선택했는데 하나 이상의 탭이나 창이 열려 있으면, &brandShortName;는 사용자가 정말로 종료하려는지 물어 보아 실수로 종료되지 않도록 할 것입니다. &pref.menuPath; 메뉴의 탭 설정에서 여러 탭을 닫을 때 확인하기 확인란의 선택을 해제하여 이 경고 메시지가 나타나지 않게 할 수 있습니다. &brandShortName;는 또한 파일을 다운로드 하고 있을 때에도 경고 메시지를 보냅니다.

편집

실행 취소

텍스트 입력란에서 행한 마지막 동작을 취소합니다. 나중에 이 실행 취소가 잘 못 되었다고 판단되면, 실행 반복을 사용합니다.

실행 반복

마지막으로 실행 취소된 명령을 다시 실행합니다.

잘라내기

텍스트 입력란에서 텍스트를 잘라내어 클립보드에 저장합니다. 붙여넣기 명령을 이용하여 다른 곳에 붙여 넣습니다.

복사

선택된 텍스트를 클립보드에 복사합니다. 붙여넣기 명령을 이용하여 해당 텍스트를 텍스트 입력란에 붙여넣습니다.

붙여넣기

클립보드에 저장된 텍스트를 텍스트 입력란에 붙여넣습니다.

삭제

선택된 텍스트를 텍스트 입력란에서 삭제합니다.

모두 선택

해당 페이지에 있는 모든 텍스트와 기타 내용물을 모두 선택합니다.

이 페이지에서 찾기…

브라우저 하단에 검색창이 나타납니다. 찾고자 하는 텍스트를 검색창에 입력하면, 입력과 동시에 &brandShortName;가 자동적으로 해당 페이지에서 검색된 첫 텍스트가 선택합니다. 아래로 또는 위로 클릭하면 페이지 내에서 다른 곳에 있는 텍스트를 검색하고, 모두 강조를 클릭하면 페이지 내에서 검색된 모든 텍스트가 강조됩니다.

다시 찾기

이 페이지에서 찾기…를 사용할 때 다음 텍스트를 검색합니다.

&pref.pluralCaps;

&pref.pluralCaps; 창에서 &brandShortName;에 있는 많은 &pref.plural;을(를) 변경할 수 있습니다.

보기

도구 모음

표준 도구 모음

표준 도구 모음을 표시하거나 숨깁니다. 이 도구 모음은 보통 표준 이동 단추(뒤로, 앞으로, 중지 등)을 담고 있습니다.

북마크 도구 모음

북마크 도구 모음을 표시하거나 감춥니다. 이 도구 모음 다양한 사이트에 대한 북마크를 담고 있습니다.

사용자 지정…

도구 모음 설정 대화 상자가 나타납니다. 도구 단추을 해당 도구 모음으로 끌어다 놓거나 끌어 올 수 있습니다. 또한 표준 도구 모음 위에서 오른쪽 클릭&ctrlKey;,클릭 하면 사용자 지정… 대화 상자가 나타 납니다.

상태 표시줄

창 하단에 있는 상태 표시줄을 표시하거나 숨깁니다. 상태 표시줄은 페이지에 대한 유용한 정보를 보여 주지만 모든 사람에게 필요한 것은 아닙니다.

탐색 창

탐색 창에는 북마크나 열어본 페이지 목록을 보여 줄 수 있습니다. 오른쪽-위 모퉁이에 있는 X를 클릭하면 언제든지 탐색 창을 닫을 수 있습니다.

북마크

북마크 창을 통해 사용자의 북마크를 항상 볼 수 있습니다. 북마크를 자주 쓰는 사용자에게 매우 유용할 수 있습니다.

열어본 페이지 목록

열어본 페이지 목록 창을 통해 방문한 페이지 목록을 항상 볼 수 있습니다. 페이지가 저정된 기간(일 수)을 변경하려면, &pref.menuPath; 메뉴를 열어 개인 정보 설정을 선택합니다.

중지

보여지고 있는 현재 페이지 부르는 것을 중지합니다. 이것은 도구 모음에 있는 중지 단추을 클릭하는 것과 동일합니다.

새로 고침

현재 페이지의 최신 상태를 불러 옵니다. 이것은 도구 모음에 있는 새로 고침 단추을 클릭하는 것과 동일합니다.

텍스트 크기

웹 페이지에 있는 텍스트가 너무 작거나 크면, 좀더 적당한 크기로 변경할 수 있습니다.

확대

웹 페이지에 있는 텍스트를 한 칸 확대합니다. 마우스를 이용해서 수행할 수도 있습니다(마우스 단축키 참조).

축소

웹 페이지에 있는 텍스트를 한 칸 축소합니다. 이는 마우스를 이용해서 수행할 수도 있습니다(마우스 단축키 참조).

보통

웹 페이지에 있는 텍스트를 보통 크기로 보여 줍니다.

페이지 스타일

현재 웹 페이지를 위한 스타일을 선택할 수 있습니다. 웹 제작자가 달리 기본 스타일을 지정하지 않았다면, "기본 페이지 스타일"이 선택됩니다.

스타일 없음

현재 페이지에서 구조화된 스타일을 제거합니다.

기본 페이지 스타일

웹 제작자가 지정한 일반적 스타일을 적용하여 페이지를 보여 줍니다.

웹 제작자가 지정한 추가적 스타일이 있으면, &brandShortName;는 사용자가 선택할 수 있도록 목록을 보여 줄 것입니다.

문자 인코딩

해당 웹 페이지의 문자 인코딩을 수동으로 변경할 수 있습니다. 보통 &brandShortName;가 자동으로 이를 수행합니다.

소스

현재 문서의 소스 코드를 보여 줍니다.

전체 화면

창을 전체 화면으로 보여 줍니다. 표준 도구 모음이 보이지 않는 상태에서, F11를 누르면 언제든지 원래 창으로 되돌아 올 수 있습니다.

바로 가기

바로 가기 메뉴에는 최근 열어본 웹 사이트와 닫은 탭 목록이 있습니다.

뒤로

열어본 페이지 목록에서 한 페이지 뒤로 이동합니다. 도구 모음에서 뒤로 단추을 클릭한 것과 동일 합니다.

앞으로

열어본 페이지 목록에서 한 페이지 앞으로 이동합니다. 이번에는 도구 모음에서 앞으로 번튼을 클릭한 것과 동일합니다.

홈 페이지

사용자의 홈 페이지를 보여 줍니다. 표준 도구 모음에서 홈 단추을 클릭한 것과 동일합니다. 홈 페이지를 바꾸려면, &pref.menuPath; 메뉴를 선택하여 일반 설정을 엽니다. 빈 페이지, 현재 페이지 (또는 탭 그룹), 북마크를 선택할 수도 있고, 직접 주소를 입력할 수도 있습니다.

최근에 닫은 탭

현재 창에서 최근에 닫은 탭을 복원하려면 목록에서 해당 탭을 선택합니다. 더 자세한 정보는 탭 닫기 및 복원하기 주제를 참조합니다.

탐색 창에서 보기

열어본 페이지 목록 창을 통해 방문한 페이지 목록을 항상 볼 수 있습니다. 페이지가 저정된 기간(일 수)을 변경하려면, &pref.menuPath; 메뉴를 열어 개인 정보 설정을 선택합니다.

북마크

이 메뉴에는 사용자가 저장한 모든 북마크가 있습니다.

북마크에 추가

현재 페이지를 북마크의 추가 합니다. 대화 상자에서 북마크 제목를 입력하고 이 북마크가 저장될 위치를 지정할 수 있습니다.

현재 페이지 구독하기…

현재 페이지에서 제공하는 RSS 피드 페이지의 미리보기를 보여 줍니다. 이 미리보기 페이지에서 라이브 북마크, 또는 사용자 컴퓨터에 있는 피드 리더기 또는 웹 서비스를 이용한 피드를 구독 설정 할 수 있습니다. &pref.menuPath; 메뉴 RSS 피드 설정에서 라이브 북마크나 피드 리더기를 이용하도록 설정하면, 미리보기 페이지가 나타나지 않고 바로 다음으로 넘어갈 것입니다.

모든 탭을 북마크에 추가…

현재 창에 있는 모든 탭을 새 북마크 폴더에 추가합니다. 대화 상자에서 폴더 이름을 입력하고 이 폴더가 저정될 위치를 지정할 수 있습니다.

북마크 구성…

북마크 관리자 창에서 북마크를 수정할 수 있습니다. 북마크에 대한 정렬, 이름 바꾸기와 등록 정보 변경과 더불어 추가, 삭제 및 이동을 수행할 수 있습니다.

도구

웹 검색

오른쪽 상단 웹 검색 창으로 커서가 이동합니다. 이제 원하는 단어를 입력하고 웹에서 검색할 수 있습니다.

다운로드

다운로드 관리자 창에서 완료된 다운로드는 물론 현재 진행 중인 다운로드를 볼 수 있습니다.

부가 기능

부가 기능 관리자에서 확장 기능과 테마를 보고, 설치하고, 설정하고, 업데이트하고, 설치 제거할 수 있습니다. 더 자세한 정보는 부가 기능 도움말 주제를 참조합니다.

오류 정보

오류 정보 대화 상자에서 자바스크립트 코드와 관련된 문제를 살펴 볼 수 있습니다. 자바스크립트는 웹 사이트를 구축하는데 흔하게 쓰이는 스크립트 언어입니다. 프로그래머는 자바스크립트를 이용하여 더욱 상호운용적인 웹 페이지를 만들 수 있습니다; 자바스크립트는 보통 동적으로 폼의 유효성을 검사하고 단추을 선택하는데 사용됩니다.

페이지 정보

문서의 형식, 인코딩, 크기, 그리고 보안 정보와 같은 현재 웹 페이지에 대한 광범위한 정보를 보여 줍니다. 또한 이 대화 상자는 해당 페이지에 사용된 미디어와 링크 목록도 보여 줍니다.

개인 정보 초기화…

&pref.menuPath; 메뉴의 개인 정보 설정에 있는 개인 정보 초기화 설정 대화 상자에서 선택된 항목을 초기화 합니다. 기본적으로 확인 창이 나타납니다.

&pref.pluralCaps;

&pref.pluralCaps; 창에서 &brandShortName;의 다양한 &pref.plural; 항목을 변경할 수 있습니다.

도움말

도움말 목차 &brandShortName; 도움말

웹 탐색에 도움을 줄 유용한 정보를 담고 있는 &brandShortName; 도움말 창을 엽니다.

Internet Explorer 사용자

이 도움말 창이 열리고 Internet Explorer에서 &brandShortName;로 전향한 사용자에게 도움을 줄 정보를 표시합니다.

온라인 제품 소개서

설치, 설치 제거 및 설정에 대한 정보와 더불어 기타 중요한 정보와 최신 소식도 제공합니다. 인터넷에 접속되어 있어야만 합니다.

웹 사이트 문제 보고…

&brandShortName;에서 정상적으로 작동하지 않는 웹 사이트를 &brandShortName; 개발자에게 보고합니다.

위조 사이트 보고…

위조나 "피싱"이 의심되는 웹 사이트를 보고합니다. &brandShortName;에서 피싱 보호는 Google이 제공하며, 사용자가 제출한 정보는 Google로 보내져 이 사양을 개선하는데 사용될 것입니다. 리포트는 Google의 개인 정보 보호 정책에 따라 무기명으로 제출될 것입니다. 이 기능은 인터넷 접속이 활성화되어 있어야 합니다.

업데이트 확인…

&brandShortName;에 대한 업데이트가 있는지 확인하여 다운로드 할 것이 있으면 다운로드 할 것인지를 묻는 대화 상자가 나타납니다. 업데이트가 다운로드 되면, &brandShortName;를 재시작하여 업데이트를 설치할 것인지를 묻습니다. 이 메뉴 항목은 다운로드가 완료 되었거나 설치할 준비가 되었거나 하는 상황 맞게 바뀔 것입니다.

&brandFullName; 정보

대화 상자에 현재 버전과 간략한 공헌자 정보를 포함한 &brandShortName;에 대한 정보가 나타납니다.
