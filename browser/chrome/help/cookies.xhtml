쿠키 관리하기

이 문서는 쿠키가 무엇이고, 어떻게 이용하고, 또 사용자의 컴퓨터에 저장된 쿠키를 &brandFullName; 쿠키 관리자를 이용하여 제어하는 방법을 설명합니다.

쿠키란?

쿠키는 인터넷 사이트에 의해 생성되는 파일로 사이트를 방문할 때 사용자의 정보를 사용자의 컴퓨터에 저장합니다. 쿠키를 사용하는 사이트를 방문하면, 이 사이트는 &brandShortName;에게 쿠키를 사용자의 하드 디스크에 저장할지를 물어올 것입니다.

나중에, 이 사이트를 재방문하게 되면, &brandShortName;는 해당 사이트에 속한 쿠키를 되돌려 줍니다. 이를 통해 이 사이트는 사용자의 필요에 맞게 설정된 정보를 사용자에게 보여주게 됩니다.

쿠키는 또한 개인적인 신원을 확인할 수 있는 정보를 저장할 수도 있습니다. 개인적인 신원을 확인할 수 있는 정보란 사용자의 신원을 확인하거나 연락을 취할기 위하여 이용될 수 있는 정보로, 이름, 전자 메일 주소, 가정이나 직장 주소, 또는 전화 번호와 같은 정보를 말합니다. 하지만, 웹 사이트는 사용자가 제공한 개인 정보에만 접근할 수 있습니다. 예를 들어, 웹 사이트는 사용자의 전자 메일 주소가 무엇인지 사용자가 직접 제공하지 않는 한 알 수 없습니다. 또한 웹 사이트는 사용자 컴퓨터에 있는 다른 정보에 대한 접근 권한은 획득할 수 없습니다.

기본 쿠키 설정을 그대로 사용하고 있다면, 쿠키의 작동 상황이 보이지 않으며 사용자는 언제 웹 사이트가 쿠키를 저장하는지 또는 언제 &brandShortName;가 해당 웹 사이트에게 쿠키를 보내는지 알지 못할 것입니다. 하지만, 쿠키 &pref.plural;을 통하여 사용자에게 물어 보고 쿠키를 저장토록 할 수 있습니다. 또한 현재 &brandShortName; 세션에 대한 쿠키의 수명을 제한할 수 있습니다.

쿠키 규정 설정하기

기본적으로 &brandShortName;는, 한 웹 사이트가 사용자를 사실상 영구적으로 인식할 수 있도록 하는 쿠키를 포함하여, 모든 쿠키를 수용합니다. 만일 이게 적합치 않다고 여긴다면, &brandShortName;에게 모든 쿠키를 지우고 종료되도록 알려 주어야 합니다:

  1. &pref.menuPath; 메뉴를 선택합니다.
  2. 개인 정보를 클릭합니다.
  3. 유지 기간 섹션 콤보 상자에서 &brandShortName;를 닫을 때 &pref.singular;을 선택합니다.

사용자가 신뢰하는 사이트에 대해 쿠키를 계속해서 저장할 수 있는 권한을 부여하려면 (예, 특정 사이트에 자동 로그인 허용), 예외 목록을 클릭하여, 해당 사이트 주소를 입력하고 허가을 클릭합니다.

설정 항목

&brandShortName;가 쿠키를 어떻게 관리할 것인지에 영향을 주는 &pref.plural;은 다음과 같습니다:

웹 사이트에서 제공하는 쿠키 허용

어떠한 웹 사이트도 사용자의 컴퓨터에 쿠키를 저장하기를 원치 않는다면, 이 &pref.singular;을 선택 해제합니다. 어떤 사이트는 쿠키 사용이 허용되지 않으면 제대로 작동하지 않을 수도 있음을 유념해야 합니다. 특정 사이트에 대해 쿠키를 저장할 수 있는 권한을 부여하려면 예외 목록을 클릭하여 예외 목록 창을 열고, 해당 사이트 주소를 입력하고, 허가 또는 세션 허가를 클릭합니다.

유지 기간:
  • 만료 될 때까지: 이 &pref.singular;을 선택하면, 각 쿠키는 만료 날짜가 되면 삭제될 것입니다.
  • &brandShortName; 닫을 때까지: 이 &pref.singular;을 선택하면, 쿠키는 &brandShortName;를 재시작할 때 삭제될 것입니다. 쿠키 사용을 요구하는 웹 사이트는 문제 없이 작동은 할 것이지만, 이 웹 사이트는 사용자가 처음으로 접속하는 것으로 인식할 것입니다.
  • 매번 물어 보기: 웹 사이트가 쿠키 저장을 시도할 때마다 대화 상자가 나타나서 해당 쿠키를 저장할 것인지를 물어올 것입니다.

쿠키 허용하기 및 차단하기

유지 기간 콤보 상자에서 매번 물어 보기를 선택하면, 사이트가 사용자의 컴퓨터에 쿠키를 저장하려고 시도할 때마다 다음과 같은 선택 사항을 묻는 대화 상자가 나타날 것입니다:

허가
특정 쿠키를 허용하려면, 단순히 허가를 클릭합니다. 이것은 사이트를 신뢰하는 경우 사용합니다.
세션 허가
지금은 허용을 하지만, &brandShortName;를 종료할 때 지우기를 원한다면, 세션 허가를 클릭합니다. 이것은 사이트가 쿠키 없이는 작동하지 않지만 쿠키를 영구적으로 저장하기를 원치 않는 경우 사용합니다.
거부
쿠키 저장을 원치 않으면, 거부를 클릭합니다. 이것은 사이트를 신뢰하지 않거나 사이트가 사용자의 개인 정보를 위태롭게할 것으로 의심되는 경우 사용합니다.
이 사이트의 모든 쿠키에 설정 적용
위 단추 중 하나를 선택하기 전에 이 확인 상자를 선택하면, &brandShortName;가 사용자가 선택한 것을 기억했다가 다시 묻지 않습니다. 해당 사이트가 쿠키 예외 목록 창에 추가될 것이며 거기서 나중에 선택 사항을 바꿀 수 있게 됩니다.

쿠키 예외 목록 창

이 창을 열려면 쿠키 섹션에서 예외 목록 단추를 클릭합니다. 여기서 특정 사이트에 대한 예외 목록을 만들 수 있습니다. 예외 목록을 이용하면, 모든 쿠키를 허용할 수도 있고, 모든 쿠키를 거부할 수도 있고, 아니면 허용은 하지만 &brandShortName; 닫으면 모두 지워지게 할 수도 있습니다. 이 예외 설정은 다른 쿠키 설정과 상관없이 작동합니다.

사이트를 목록에 추가하려면, 간단히 사이트의 도메인명을 웹 사이트 주소 입력합니다. 차단을 클릭하여 해당 사이트로터의 쿠키를 거부하거나, 허가를 클릭하여 해당 사이트로부터의 쿠키를 허용하거나, 또는 세션 허가를 클릭하여 해당 사이트로부터의 쿠키를 허용은 하지만 &brandShortName;를 닫으면 지워지게 할 수 있습니다.

이 예외 목록에서 사이트를 제거하려면, 제거를 클릭합니다. 목록 전체를 완전히 지우려면, 모두 제거를 클릭합니다. 그러면 예외 목록이 초기화 되어 일반 쿠키 &pref.plural;이 적용됩니다.

쿠키 보기 창

쿠키 보기 설정을 이용하여 쿠키를 보고 삭제할 수 있습니다. 이 창을 열려면 쿠키 &pref.plural; 섹션에서 쿠키 보기 단추를 클릭합니다.

쿠키 보기 창은 현재 사용자의 컴퓨터에 저장되어 있는 모든 쿠키를 사이트 별로 묶어서 보여 줍니다. 사이트를 선택하여 쿠키 목록을 펼쳐서 쿠키를 선택하면 좀 더 자세한 정보가 표시됩니다. 또한 사이트나 쿠키 이름을 검색 창에 입력하여 검색할 수도 있습니다.

목록에서 쿠키를 삭제하려면, 해당 쿠키를 선택하고 삭제를 클릭합니다. 사이트의 모든 쿠키를 삭제하려면, 해당 사이트를 선택하고 삭제를 클릭합니다. 모든 쿠키를 완전히 삭제하려면, 모두 삭제를 클릭합니다.
