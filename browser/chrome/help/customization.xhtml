&brandFullName; 사용자 설정하기

세부 항목:
  • 도구 모음
  • 부가 기능 (확장 기능과 테마)

도구 모음

이 문서는 &brandShortName;에서 도구 모음을 재배치하고, 숨기고, 추가하고, 삭제하는 방법을 설명할 것입니다. 용어에 익숙하지 않은 사용자를 위해 설명하자면, 도구 모음이란 웹 사이트가 표시되어 있는 영역 위 쪽에 있는 (보통은 회색) 상자를 말합니다.

  • 메뉴 도구 모음: 브라우져 메뉴(파일, 편집, 도움말 등)을 담고 있는 도구 모음입니다.
  • 표준 도구 모음: 주소 표시줄과 웹 사이트 탐색에 이용되는 단추를 담고 있는 도구 모음입니다.
  • 북마크 도구 모음: 북마크 도구 모음 폴더의 북마크를 담고 있는 도구 모음입니다.

&brandShortName;에서 도구 모음을 사용자 설정하는 기초적인 방법은 두가지가 있습니다:

  • 도구 모음 항목 재배치
  • 도구 모음을 숨기고, 추가하고, 삭제하기

도구 모음 항목 변경하기

도구 모음에 있는 항목(아이콘, 단추, 텍스트 상자, 등)을 변경하려면, 도구 모음에서 텍스트 상자가 아닌 곳 아무데나에서 오른쪽 클릭&ctrlKey;를 누르고 마우스 클릭하고 사용자 지정…을 선택합니다. 도구 모음 설정 대화 상자가 나타날 것입니다.

항목을 끌어다 놓기로 대화 상자에서 도구 모음으로 추가하거나 도구 모음에서 삭제합니다. 이미 도구 모음에 존재하는 아이콘을 원하는 곳으로 끌어다 놓아 배치를 바꿀 수도 있습니다. 도구 모음과 항목이 원하는 대로 배치되면, 확인을 클릭합니다.

도구 모음 특수 항목

도구 모음을 사용자 설정할 때 특수한 항목이 몇 개 있는데, 다음과 같습니다:

  • 구분자: 도구 모음에 있는 항목을 구분하는 작은 수직선 입니다.
  • 공백: 도구 모음에 있는 항목을 구분하는 고정된 크기의 공백으로, 대부분의 테마에서 보통 20 픽셀정도 입니다.
  • 가변 공백:도구 모음에 있는 항목을 구분하는 공백으로, 이 공백은 자동적 도구 모음의 빈 공간이 꽉 차게 크기가 변경됩니다. 이것은 항목을 도구 모음의 오른 쪽에 위치시킬 때 유용합니다.

도구 모음 숨기기, 추가하기 및 삭제하기

표준 도구 모음과 북마크 도구 모음이 화면에 계속 표시되어 있는 것을 원치 않으면 숨길 수 있습니다. 메뉴 도구 모음을 숨길 수는 없는데 이는 메뉴 항목들(파일, 편집, 도움말 등)을 다른 도구 모음으로 이동시킬 수 없기 때문이며, 이 항목들은 항상 화면에 표시 되어 있어야 합니다.

도구 모음을 표시하거나 숨기려면, 보기 > 도구 모음 을 선택하고 표시하거나 숨기를 원하는 도구 모음 이름을 클릭합니다.

사용자 도구 모음을 추가하려면, 보기 > 도구 모음 > 사용자 지정을 선택합니다. (대화 상자 하단에 있는) 새 도구 모음 추가를 클릭합니다. 도구 모음의 이름을 입력할 수 있는 대화 상자가 나타날 것입니다. 이름을 입력하고 확인을 클릭하면, 새로운 도구 모음이 나타날 것입니다. 이제 항목을 이 새로 만든 도구 모음에다 끌어다 놓습니다. 사용자 도구 모음을 제거하려면, 그 안에 있는 모든 항목을 끌어 내고, 확인을 클릭합니다. 사용자 도구 모음 또한 이전에 설명한 것과 같은 방법으로 숨길 수 있습니다.

부가 기능 (확장 기능과 테마)

부가 기능은 &brandShortName;의 외양이나 기능을 변경하거나 추가하는 작은 크기의 소프트웨어입니다. 부가 기능의 종류에는 확장 기능과 테마라는 두가지가 있습니다.

확장 기능
확장 기능은 &brandShortName;에 새로운 기능을 추가합니다. 확장 기능은 도구 모음 단추하나를 추가하는 것에서부터 완전히 새로운 사양을 추가하는 것까지 다양합니다. 확장 기능은 응용프로그램 자체의 크기와 외양을 최소화하면서 각 사용자의 개인적인 필요를 충족시킬 수 있도록 합니다.
테마
테마는 &brandShortName;의 외양을 바꿉니다. 테마는 &brandShortName;의 외관과 느낌을 바꾸어 사용자의 구미에 맞게 변경합니다. 테마는 단순히 단추 이미지일 수도 있고, &brandShortName;의 모든 외관을 바꿀 수도 있습니다.

부가 기능 구하기 및 설치하기

많은 부가 기능을 &brandShortName; Add-ons에서 설치할 수 있습니다. 또한 인터넷 검색 통해 또한 다른 많은 확장을 찾을 수 있습니다.

확장 기능을 설치하기 위해 링크를 클릭하면 &brandShortName;는 대화 상자를 표시하고 이 부가 기능 설치를 허용할 것인지를 물어옵니다. 다운로드를 허용하고 설치하도록 할 수도 있고 절차를 중단할 수도 있습니다.

부가 기능 관리자 사용하기

부가 기능 관리자에서 부가 기능을 관리합니다. 도구 > 부가 기능를 선택하여 관리자를 엽니다.

부가 기능 업데이트

기본적으로 &brandShortName;는 설치한 부가 기능에 대한 새 버전이 있는지를 주기적으로 검사할 것입니다. 새 버전이 있으면, 다음 시작 시 업데이트를 설치할 것인지를 물을 것입니다. 이런 동작 방식은 &pref.pluralCaps; 메뉴 고급 설정있는 업데이트 탭에서 변경할 수 있습니다.

수동으로 업데이트를 검사하려면, 확장 기능이나 테마를 선택하고 업데이트 찾기 단추를 클릭합니다. 업데이트가 있으면 부가 기능 관리자에 업데이트 단추가 표시되고 설치하고자 하는 업데이트를 선택할 수 있습니다. 업데이트 설치 단추를 누르면 행당 업데이트가 설치됩니다. &brandShortName;를 재시작해야만 변경사항이 적용 됩니다.

부가 기능 사용 안함, 사용함, 설치 제거하기

부가 기능을 제거하려면, 임시적으로 사용 중지하여 나중에 쉽게 다시 사용할 수 있게 할 수도 있고 설치 제거할 수도 있다. 테마는 설치 제거만 할 수 있는데, 이는 어차피 현제 사용 중인 테마 외에는 모두 사용 중지 되어 있기 때문입니다.

확장 기능을 사용하든지 사용 중지하려면, 원하는 확장 기능을 선택하고 사용 안 함이나 사용함 단추를 클릭합니다. &brandShortName;에서 부가 기능을 제거하려면, 원하는 부가 기능을 선택하고 설치 제거 단추를 클릭합니다. &brandShortName;를 재시작해야만 변경 사항이 적용됩니다.

기타 기능

홈 페이지 방문이나 정보와 같은 기타 기능은 해당 부가 기능 위에서 &ctrlKey;를 누르 오른쪽 클릭합니다.

확장 기능과 테마관련 작업

확장 기능 설정

확장 기능은 보통 기능을 설정할 수 있는 대화 상자를 제공합니다. 확장 기능의 설정 대화 상자를 열려면, 확장 기능 탭을 선택하고 원하는 확장 기능을 선택한 후, &pref.pluralCaps; 단추를 클릭하거나 간단히 해당 확정 기능을 더블 클릭 합니다.

테마 바꾸기

설치된 테마 중 사용할 테마를 바꾸려면, 테마 탭을 선택하고, 원하는 테마를 선택한 후, 테마 사용을 클릭합니다. &brandShortName;를 재시작 해야만 변경 사항이 적용될 것입니다.
